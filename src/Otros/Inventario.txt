Elaborar un programa para leer el nombre, la cantidad y el precio de un determinado número de productos de una tienda de autoservicio y manipular las siguientes opciones, a través de un menú.
1. Insertar datos.
2. Listar datos.
3. Modificar precio
4. Consultar producto
5. Entrada de producto
6. Salida de producto
7. Inventario
Salir

Opciones:
1.       Esta opción permite insertar datos. (nombre, precio y cantidad)
2.       Esta opción debe imprimir los datos incluyendo el total.
3.       Esta opción debe permitir modificar el precio.
4.       Esta opción envía la cantidad y el precio de un producto en específico. Ejemplo
            Mayonesa 300 grs Cantidad 45, precio $32.000
5.       Esta opción modifica la cantidad de un producto específico sumando una cantidad determinada.
6.       Esta opción modifica la cantidad de un producto específico restando una cantidad determinada, siempre y cuando esa cantidad no sea mayor al número de los productos existentes.
7.       Esta opción me muestra el total de todos los productos, tomando el ejemplo anterior sería. 365 productos que equivales a $9,730.00
